Project Pitch

Marketing Portfolio Website

A personal portfolio website created to present my work in social media management, content creation, and campaign planning — and to serve as a strong base for my future clients.

Live Website GitHub Repo

Layout & Structure

The website is built with a clear section-based structure to make it easy to expand over time. Styling is consistent using shared components and variables.

  • Section-based layout (About, Work, Contact)
  • Reusable “glass card” components
  • Consistent spacing + typography system
  • CSS variables for colours and radiuses

Interactivity (JavaScript)

JavaScript is used only where needed — for cookie consent logic and contact form validation feedback.

  • Cookie consent (Accept / Decline)
  • Choice stored using localStorage
  • Client-side form validation using HTML validity checks
  • Clear feedback message after submission

Navigation & Responsiveness

Navigation uses Bootstrap behaviour with custom styling, including dropdown links and a mobile menu.

  • Sticky navigation bar
  • Dropdown menu for “Expertise”
  • Mobile hamburger menu
  • Responsive adjustments via media queries

Validation & Standards

The final HTML was checked using W3C Validator and cleaned up before submission.

  • HTML validated using W3C Validator
  • Accessibility-friendly structure (labels, aria where needed)
  • Clean, readable semantic layout
